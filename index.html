One-time Supabase setup

1
Create a free Supabase account
Go to supabase.com → click "Start your project" → sign up with GitHub or email. It's free.
2
Create a new project
Click "New project" → name it daily-brief → pick US East → set a database password → Create.
3
Run this SQL in the SQL Editor
In your project → SQL Editor → paste and run:
CREATE TABLE brief (id text PRIMARY KEY, data jsonb);
ALTER TABLE brief ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public read" ON brief FOR SELECT USING (true);
CREATE POLICY "public write" ON brief FOR ALL USING (true) WITH CHECK (true);
4
Copy your credentials
Project Settings → API → copy the Project URL and publishable (anon) key. Paste below.
